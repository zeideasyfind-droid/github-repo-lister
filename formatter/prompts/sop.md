## Purpose
This SOP ensures the formatter produces consistent, high-quality property listings.

## Reference
The authoritative SOP is located at:
**`docs/EasyFind_Property_Formatter_SOP.md`**

## Key Principles
1. **Accuracy**: Never invent or guess information
2. **Consistency**: Always apply the same rules
3. **Validation**: Verify output before returning
4. **Google Maps**: Use as authoritative source for location data
5. **Standardization**: Apply all formatting rules uniformly
